No Man's Sky Black Hole Suns Project

Use this site as input for the Black Hole Suns router. (need link).

Player Name - Name associated with your game and with your input here. It will show up in the statistics information.

Platform & Galaxy: Different platforms and different galaxies do not have the same information.

Black Hole System, Exit System, Single System

Address: NMS Galactic Address formatted "0000:0000:0000:0000". you can get this from a signal booster on any planet. This will be reformatted into a correct address when you leave the entry field. Anything other than the characters "0123456789abcdef" count as separators. A string with no separators will be broken up into blocks of 4 characters each. All black hole system addresses end with "0079".

System name and Region name: These are from the galactic map. If you renamed the system be sure to use the new name. Otherwise, someone who tries to use it later won't be able to find it.

Lifeform and Economy: These are from the galactic map and can also be found on the discoveries page.

All black hole systems must be entered in pairs.

Systems with base computer and teleporter: You can enter systems where you have or have visited a built base with a base computer and teleporter. It will only work if it shows up in your teleport list. The router can use these as your first move if you have them available. But this is not required for the router to operate. It can just generate faster routes if you have them available. If you entered a space station you've visited then the station can fall off of your transporter list and it wouldn't be available to you anymore. These are only available for the current user.
